
# Portfolio Project

## fetch Api

This project is built to help people to know more about **Crypto Currency**.

As we all know the 21st century is a digital and technologic era . The creation of digital or virtual money had created an other way for people to buy and trade.
The Portfolio Project will be one of the application where people can learn what the virtual money is and all the different kind and how to use it. 
A fetch Api was used to develop this project.


## Deployment

To deploy this project run

```bash
  npm run deploy
```


## Installation

Install portfolio project with npm

```bash
  npm install My_Portfolio_Project
  cd my-project
```
